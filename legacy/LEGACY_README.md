# Legacy Codebase

This directory contains the original ServiceBridge codebase (circa 2023), archived during the 2026 Phase 2 Modernization Rebuild.

## Contents Archived
- Pure HTML marketing pages (`index.html`, `about.html`, `services.html`, etc.)
- Authentication endpoints relying on client-side Auth0
- Hardcoded payment instructions script (`payment-script.js`)
- UI utilizing Bootstrap and jQuery

## Preserved Assets
- All images are preserved in `assets/images/`
- Brand copy and marketing text can be referenced from the HTML templates

This code is strictly retained for reference and is not part of the active production build.
